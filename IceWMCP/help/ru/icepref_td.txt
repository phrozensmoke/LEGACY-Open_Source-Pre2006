[PHTITLE] IceWMCP IcePref2: Дизайнер тем
[PHSECTION] Обзор
IceWMCP IcePref2: Дизайнер тем позволяет вам легко создать и изменить темы IceWM. В данном руководстве освещены наиболее важные аспекты использования Дизайнера тем IcePref2.
[PHSECTION] Дизайнер тем IcePref2
В главном окне Дизайнера тем IcePref2 представлены всевозможные компоненты тем IceWM,  которые распределены по [PH-HI]разделам[/PH-HI], таким как 'Поле адреса', 'Меню', 'Мышь' и т.д. Для быстрого перехода из одного раздела в другой, выберете нужный пункт из выпадающего списка [PH-HI]Раздел[/PH-HI] в главном меню Дизайнера тем. [PH-HI]Чтобы приступить к работе с помощью Дизайнера, вы должны либо открыть уже существующую тему, либо создать новую[/PH-HI].
[PHSECTION] Открытие файлов тем
Чтобы [PH-HI]открыть[/PH-HI] для правки файл уже существующей темы IceWM, выберете [PH-HI] Файл -> Открыть тему... [/PH-HI] в главном меню, или нажмите [PH-HI]Ctrl+O[/PH-HI] на клавиатуре. Укажите тему, которую хотите открыть, а затем щелкните на кнопке [PH-HI]'OK'[/PH-HI].
[PHSECTION] Создание новой темы
Для [PH-HI]создания[/PH-HI] новой темы IceWM, выберете [PH-HI] Файл -> Новая тема... [/PH-HI] в главном меню, или нажмите [PH-HI]Ctrl+N[/PH-HI] на клавиатуре. Откроется новое диалоговое окно, в котором вы должны заполнить необходимые поля: введите будущее [PH-HI]название[/PH-HI] темы в поле [PH-HI]Название темы:[/PH-HI], введите [PH-HI]ваше имя[/PH-HI] (или псевдоним) в поле [PH-HI]Автор темы:[/PH-HI]. Если желаете, можете тут же указать ваш адрес электронной почты. Также введите [PH-HI]короткое описание[/PH-HI] создаваемой темы в поле [PH-HI]Описание темы:[/PH-HI]. После чего остается указать [PH-HI]каталог[/PH-HI], куда тема будет сохранена. Для этого щелкните на кнопке рядом с полем [PH-HI]Директория:[/PH-HI]. Пожалуйста, убедитесь, что у вас есть право на доступ для записи файлов в каталог, который вы указали. И наконец щелкните на кнопке [PH-HI]'OK'[/PH-HI]. Создастся новая тема.
[PHSECTION] Сохранение темы
Перед выходом из программы, вы должны убедиться, что [PH-HI]сохранили[/PH-HI] все проделанные с помощью Дизайнера тем изменения. Чтобы сохранить новую тему IceWM,  выберете [PH-HI] Файл -> Сохранить тему [/PH-HI] в главном меню, или нажмите [PH-HI]Ctrl+S[/PH-HI] на клавиатуре. Другой способ проделать это - щелкнуть на кнопке [PH-HI]Сохранить[/PH-HI] в нижней части экрана Дизайнера тем IcePref2.
[PHSECTION] Инструменты
Для настройки IceWM вы можете воспользоваться дополнительными инструментами, щелкнув на кнопке [PH-HI] Инструменты [/PH-HI] в главном меню Дизайнера тем. Выберете нужный инструмент из выпавшего списка.
[PHSECTION] Выход из программы
Чтобы выйти из программы, щелкните [PH-HI] Файл -> Выход из Дизайнера...[/PH-HI] в главном меню, или нажмите[PH-HI] Ctrl+Q [/PH-HI] на клавиатуре. Выйти из программы также можно щелкнув на кнопке [PH-HI]Выход[/PH-HI], которую вы можете найти справа, в нижней части экрана Дизайнера тем IcePref2.
[PHSECTION] Поиск новых версий программы
Для поиска новых версий программы выберете [PH-HI] Файл -> Проверка новых версий программы... [/PH-HI] в главном меню, или нажмите [PH-HI] Ctrl+U [/PH-HI] на клавиатуре. Для этого необходимо установленное соединение с Интернет.
[PHSECTION] Запуск произвольных программ
Вы можете выполнить любую программу в вашей системе, выбрав [PH-HI] Файл -> Запуск... [/PH-HI] в главном меню, или, нажав [PH-HI] Ctrl+R [/PH-HI] на клавиатуре.
[PHSECTION] Сообщения автору об ошибках в программе
Вы можете сообщить автору об ошибках, выбрав в главном меню IceMe [PH-HI] Помощь -> Отправить отчет об ошибке... [/PH-HI], или нажмите [PH-HI] F5 [/PH-HI] на клавиатуре. Вы увидите подробную инструкцию, как пройти эту короткую процедуру.
