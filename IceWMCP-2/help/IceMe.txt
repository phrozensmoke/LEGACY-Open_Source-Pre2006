[PHTITLE] IceWMCP IceMe
[PHSECTION] Overview
The IceWMCP IceMe tool allows you to easily edit your IceWM menu and toolbar and configure the look and behavior of your IceWM menus.  This help text includes only the most basic information.  All buttons in the IceMe program have popup [PH-HI]'hints'[/PH-HI], which you can see by placing your mouse cursor over any button.  These [PH-HI]'hints'[/PH-HI] provide the majority of the information you need for more advanced usage.  You should pay close attention to the collection of buttons at the top of the IceMe window.


[PHTITLE] IceMe : Menu
Click the [PH-HI]Menu[/PH-HI] tab. The main window of [PH-HI]IceMe : Menu[/PH-HI] module is divided into two sections.  On the left, is the [PH-HI]menu 'tree'[/PH-HI] containing IceWM menu.  On the right, is the [PH-HI]'Menu entry'[/PH-HI] area for editing items on the [PH-HI]menu 'tree'[/PH-HI].
[PHSECTION] Adding A Menu Item
A [PH-HI]'menu item'[/PH-HI] is a single program on the menu 'tree'.  To add a new menu item, open the [PH-HI]folder[/PH-HI] on the menu 'tree' where you wish to add the new item.  You can open the folder by either expanding the menu 'tree' or by double-clicking on the folder.  Click the button [PH-HI]'Create a new normal entry'[/PH-HI] at the top of the IceMe window.  (The button contains a picture of a 'blank' sheet of paper.)   In the [PH-HI]'Menu entry'[/PH-HI] area, on the right side of the IceMe window, complete the requested information about your new 'menu item'.  Provide a name for the program in the [PH-HI]Name:[/PH-HI] area.  Specify the executable to be launched in the [PH-HI]Command:[/PH-HI] area, or click the [PH-HI]'folder'[/PH-HI] button.  Finally, select an icon for the 'menu item' in the [PH-HI]Icon:[/PH-HI] area, or click the [PH-HI]'folder'[/PH-HI] button.
[PHSECTION] Adding A Menu Folder
A [PH-HI]'menu folder'[/PH-HI] contains one or more [PH-HI]menu items[/PH-HI].  To add a new menu folder, click the button [PH-HI]'Create a new submenu'[/PH-HI] at the top of the IceMe window.  (The button contains a picture of a 'menu'.)   In the [PH-HI]'Menu entry'[/PH-HI] area, on the right side of the IceMe window, provide a name for the new folder in the [PH-HI]Name:[/PH-HI] area.
[PHSECTION] Adding A Menu Separator
A menu [PH-HI]'separator'[/PH-HI] is a straight, horizontal line within the menu, generally used to create space or to aid with menu organization.  To add a new menu separator, open the folder on the menu 'tree' where you wish to add the new separator.  You can open the folder by either expanding the menu 'tree' or by double-clicking on the folder.  Click the button [PH-HI]'Create a new separator'[/PH-HI] at the top of the IceMe window.  (The button contains a picture of one thin, straight line.) 
[PHSECTION] Deleting A Menu Item Or Folder
To [PH-HI]delete[/PH-HI] a menu item or folder, select the item or folder you wish to delete.  Click the button [PH-HI]'Delete selected entry'[/PH-HI] at the top of the IceMe window.  (The button contains a picture of an 'X'.)  You may also delete a menu item or folder by selecting [PH-HI] Edit -> Delete [/PH-HI] on the IceMe top menu.
[PHSECTION] Copying A Menu Item Or Folder
To [PH-HI]copy[/PH-HI] a menu item or folder, select the item or folder you wish to copy.  Press [PH-HI] Ctrl+C [/PH-HI] on your keyboard, or select [PH-HI] Edit -> Copy [/PH-HI] on the IceMe top menu.
[PHSECTION] Cutting A Menu Item Or Folder
To [PH-HI]cut[/PH-HI] a menu item or folder, select the item or folder you wish to cut.  Press [PH-HI] Ctrl+X [/PH-HI] on your keyboard, or select [PH-HI] Edit -> Cut [/PH-HI] on the IceMe top menu.
[PHSECTION] Pasting A Menu Item Or Folder
To [PH-HI]paste[/PH-HI] a menu item or folder, select the item or folder where you wish to paste.  Press [PH-HI] Ctrl+V [/PH-HI] on your keyboard, or select [PH-HI] Edit -> Paste [/PH-HI] on the IceMe top menu.
[PHSECTION] Toolbar
At the bottom of the IceMe menu 'tree', you will see a folder named [PH-HI]'Toolbar'[/PH-HI].  This folder holds the configuration for your IceWM toolbar (on the IceWM taskbar) and can be edited in the same way as all other portions of the IceMe menu 'tree'.
[PHSECTION] Previewing The Menu
To preview your IceWM menu, press [PH-HI] Ctrl+P [/PH-HI] on your keyboard, or select [PH-HI] File -> Preview [/PH-HI] on the IceMe top menu.
[PHSECTION] Saving The Menu
Before exiting the program, you should be sure to save your menu.  To save your IceWM menu, press [PH-HI] Ctrl+S [/PH-HI] on your keyboard, or select [PH-HI] File -> Save [/PH-HI] on the IceMe top menu.
[PHSECTION] Run As Root
If you are the 'root' user and wish to edit the system-wide IceWM menus, select [PH-HI] File -> Run As Root [/PH-HI] on the IceMe top menu.
[PHSECTION] Open IceWM Menu File
To open an IceWM menu in ANY directory on the system, select [PH-HI] File -> Open IceWM Menu File [/PH-HI] on the IceMe top menu, or press [PH-HI] Ctrl+O [/PH-HI] on your keyboard.  Please be aware that you may need to be 'root' to open and edit some menu files.


[PHTITLE] IceMe : Behavior
Click the [PH-HI]Behavior[/PH-HI] tab. The [PH-HI]IceMe : Behavior[/PH-HI] module allows you to easily configure the look and behavior of your IceWM menus using an embedded version of the IcePref2 menu configuration tool.  This module also allows you to control what special menu items are displayed on your IceWM menus.  Change the properties as you see fit.  When you are done, click the [PH-HI]'Apply Changes Now'[/PH-HI] button at the bottom of your screen.


[PHSECTION] Exiting the Program
To exit the program, click [PH-HI] File -> Exit [/PH-HI] on the menu, or press [PH-HI] Ctrl+Q [/PH-HI] on your keyboard.
[PHSECTION] Checking for New Versions of the Software
To check for a newer version of the program click [PH-HI] File -> Check for newer versions of this program... [/PH-HI] on the menu, or press [PH-HI] Ctrl+U [/PH-HI] on your keyboard.    This feature requires a working connection to the internet.
[PHSECTION] Launching Random Programs
You may launch a random program on your system by clicking [PH-HI] File -> Run... [/PH-HI] on the menu, or press [PH-HI] Ctrl+R [/PH-HI] on your keyboard. 
[PHSECTION] Reporting Problems
You may report problems using this software by clicking [PH-HI] Help -> Send A Bug Report... [/PH-HI] on the menu, or press [PH-HI] F5 [/PH-HI] on your keyboard.  You will be shown detailed instructions to guide you through the short process.
