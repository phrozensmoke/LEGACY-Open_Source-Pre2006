[PHSECTION]Обзор:
Клавиатура IceWMCP: Позволяет легко настроить частоту и задержку повтора. Первым делом нажмите на вкладку "Повтор".
[PHSECTION]Включение/Отключение автоповтора клавиатуры:
Для включения автоповтора отметьте пункт "Разрешить автоповтор клавиатуры". Для отключения автоповтора снимите отметку с этого пункта.

[PHSECTION]Установка задержки клавиатуры:
Для этого, установите желаемую задержку на ползунке под названием Задержка:. Чем меньше число, тем короче задержка. Чем число больше, тем задержка длиннее.

[PHSECTION]Установка частоты клавиатуры:
 Для этого, установите желаемую задержку на ползунке под названием Частота:. Чем меньше число, тем ниже частота . Чем число больше, тем частота выше. 
  
[PHSECTION]Применение изменений:
Для этого, нажмите кнопку Применить.

[PHSECTION]Проверка частоты и задержки клавиатуры:
Поместите курсор мыши в поле под названием Для проверки повтора, держивайте клавишу несколько секунд:. Затем несколько секунд удержите какую-нибудь клавишу (что-нибудь вроде 'm'). Если вы недовольны зарержкой и частотой, то установите их новые значения, нажмите кнопку Применить и повторите проверку в поле Для проверки повтора, держивайте клавишу несколько секунд:.

[PHSECTION]Сброс повтора клавиатуры:
Для сброса задержки и скорости клавиатуры на разумные значения нажмите кнопку Сброс.

[PHSECTION]Обзор:
Клавиатура IceWMCP: Данный инструментарий позволяет легко настраивать сигналы и клики клавиатуры. Нажмите вкладку Звук.

[PHSECTION]Включение/Отключение клика клавиатуры:
Для включения клика отметьте пункт "Разрешить клик клавиатуры". Для отключения клика снимите отметку с этого пункта.

[PHSECTION]Установка громкости клика:
Для этого, установите желаемую громкость на ползунке под названием Громкость: в разделе Клик клавиатуры. Чем меньше число, тем меньше громкость. Чем число больше, тем громкость больше.

[PHSECTION]Включение/Отключение сигнала клавиатуры:
Для включения клика отметьте пункт "Разрешить сигнал клавиатуры". Для отключения клика снимите отметку с этого пункта.

[PHSECTION]Установка громкости сигнала:
Для этого, установите желаемую громкость на ползунке под названием Громкость: в разделе "Сигнал клавиатуры". Чем меньше число, тем меньше громкость. Чем число больше, тем громкость больше.

[PHSECTION]Установка высоты сигнала:
Для этого, установите желаемую высоту на ползунке под названием Высота: в разделе "Сигнал клавиатуры". Чем меньше число, тем ниже высота. Чем число больше, тем высота выше.

[PHSECTION]Установка продолжительности сигнала:
Для этого, установите желаемую продолжительность на ползунке под названием Продолжительность: в разделе "Сигнал клавиатуры". Чем меньше число, тем меньше продолжительность. Чем число больше, тем продолжительность больше.

[PHSECTION]Проверка сигнала клавиатуры:
Для этого нажмите кнопку Проверка.

[PHSECTION]Сброс звуков клавиатуры:
Для сброса параметров звука на разумные значения нажмите кнопку Сбросить

[PHSECTION]Обзор:
Редактирование горячих клавиш IceWMCP позволяет легко настраивать сочетания клавиш для запуска приложений. Например, вы можете сделать так, что бы программа Gimp запускалась при нажатии 'Ctrl+Alt+g'. Нажмите вкладку Горячие клавиши.

[PHSECTION]Список клавиш:
Это - окно со скроллером. Список показывает какая комбинация какие программы запускает. Комбинации расположены слева, программы - справа. 

[PHSECTION]Создание комбинаций:
Нажмите "Новая клавиша". Введите желаемую комбинацию. Например, если вы хотите использовать 'Ctrl+Alt+g', сделайте следующее: Отметьте 'Ctrl'. Отметьте 'Alt'. Наберите 'g' в поле клавиши (справа от 'Ctrl','Alt','Shift'). Затем введите название программы для этой комбинации в поле Программа:, или нажмите 'папку' для выбора. Нажмите кнопку "Добавить". Так же, нужно нажать кнопку "Сохранить". 

[PHSECTION]Редактирование комбинаций:
Выберите нужную комбинацию из "списка клавиш". Саму комбинацию ('Ctrl+Alt+g') вы можете не менять. Но можете поменять программу которую она запускает. Введите название новой программы в поле "Программа", или нажмите 'папку' для выбора. Нажмите кнопку "Установить". Так же, нужно нажать кнопку "Сохранить". 

[PHSECTION]Проверка комбинаций:
Данная фича пробует запустить назначенную программу. Для проверки выберите нужную комбинацию из списка. Нажмите кнопку "Проверка". Если программа не запускается, проверьте правильность поля "Программа".

[PHSECTION]Удаление комбинаций:
Для удаления выберите нужную комбинацию из списка. Нажмите кнопку "Удалить". Так же, нужно нажать кнопку "Сохранить".

[PHSECTION]Сохранение настроек:
Перед выходом из программы, вы должны сохранить сделанные изменения. Для этого, нажмите кнопку "Сохранить", или Ctrl+S на клавиатуре. 

[PHSECTION]Применение настроек:
Если вы хотите, что бы изменения вступили в силу немедленно, нажмите кнопку "Применить изменения", или Ctrl+A на клавиатуре. Ваши изменения будут автоматически сохранены, IceWM - быстро перезапущен, и изменения заработают.

[PHSECTION]Открытие настроек:
Для открытия файла настроек (в основном это - файл "keys"), в меню нажмите Файл->Открыть настройки горячих клавиш..., или нажмите Ctrl+O на клавиатуре.


[PHSECTION]Обзор:
Рердактирование клавиш IceWM позволяет настоить дополнительные горячие клавиши для IceWM. Нажмите вкладку "Клавиши IceWM" для для продвинутой настройки.

[PHSECTION]Выход из программы:
Для выхода из программы в меню нажмите Файл->Выход, или нажмите Ctrl+Q на клавиатуре.

[PHSECTION]Проверка новых версий программы:
Для этого в меню нажмите Файл->Проверка обновлений, или нажмите Ctrl+U на клавиатуре. Требуется рабочее соединение с Internet.

[PHSECTION]Запуск произвольной программы:
Вы можете сделать это нажав в меню Файл->Запуск, или Ctrl+R на клавиатуре.

[PHSECTION]Сообщения о проблемах:
Вы можете сделать это при помощи самой программы, нажав Помощь->Послать багрепорт или F5 на клавиатуре. Вы увидите подробные инструкции по данному процедуре (она не длинная).
