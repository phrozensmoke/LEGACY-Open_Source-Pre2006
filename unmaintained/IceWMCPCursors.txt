[PHTITLE] IceWMCP Mouse: Cursors
[PHSECTION] Overview
The IceWMCP Cursor tool allows you to configure personalized mouse cursors for use with IceWM.  First, click the [PH-HI]'Cursors'[/PH-HI] tab.
[PHSECTION] Configuring Cursors
To configure a cursor, scroll to the cursor you wish to configure. Then, drag the image you wish to use as the cursor onto the cursor's window. You may also configure a cursor by clicking the [PH-HI]'folder'[/PH-HI] button to select an image file from a file selection dialog window.  Please see the IceWM home page ([PH-HI]http://www.icewm.org[/PH-HI]) for information on acceptable cursor image formats.
[PHSECTION] Editing Cursors
To edit a cursor, scroll to the cursor you wish to edit. Then, click the  [PH-HI]'Edit'[/PH-HI] button next to the cursor.  (The [PH-HI]'Edit'[/PH-HI] button has a picture of  [PH-HI]'pencil and paper'[/PH-HI].)  The cursor image will then be opened for editing with the executable [PH-HI]'gimp-remote'[/PH-HI] if  [PH-HI]'gimp-remote'[/PH-HI] is available on your system's $PATH.  Please see the IceWM home page ([PH-HI]http://www.icewm.org[/PH-HI]) for information on acceptable cursor image formats.
[PHSECTION] Deleting Cursors
To delete a cursor, scroll to the cursor you wish to delete. Then, click the  [PH-HI]'Delete'[/PH-HI] button next to the cursor.  (The [PH-HI]'Delete'[/PH-HI] button has a picture of an [PH-HI]'X'[/PH-HI].)  
[PHSECTION] Activating A New Cursor Configuration
If you have added, modified, or deleted any cursors, you can activate your changes immediately by clicking [PH-HI] 'File' -> 'Apply Changes Now...'[/PH-HI] on the menu.  You may also activate the new changes by pressing the [PH-HI]'Apply Changes Now'[/PH-HI] button, or by pressing [PH-HI]Ctrl+A[/PH-HI] on your keyboard. IceWM will be quickly restarted, and your new cursor configuration will be activated.

[PHSECTION] Available Cursors
	[PH-HI]  left.xpm [/PH-HI]:  Left arrow
	[PH-HI]  move.xpm [/PH-HI]:  Move cursor
	[PH-HI]  right.xpm [/PH-HI]:  Right arrow
	[PH-HI]  sizeB.xpm [/PH-HI]:  Bottom resize cursor
	[PH-HI]  sizeBL.xpm [/PH-HI]:  Bottom Left resize cursor
	[PH-HI]  sizeBR.xpm [/PH-HI]:  Bottom Right resize cursor
	[PH-HI]  sizeL.xpm [/PH-HI]:  Left resize cursor
	[PH-HI]  sizeR.xpm [/PH-HI]:  Right resize cursor
	[PH-HI]  sizeT.xpm [/PH-HI]:  Top resize cursor
	[PH-HI]  sizeTL.xpm [/PH-HI]:  Top Left resize cursor
	[PH-HI]  sizeTR.xpm [/PH-HI]:  Top Right resize cursor

[PHSECTION] Exiting the Program
To exit the program, click [PH-HI] File -> Exit [/PH-HI] on the menu, or press [PH-HI] Ctrl+Q [/PH-HI] on your keyboard.
[PHSECTION] Checking for New Versions of the Software
To check for a newer version of the program click [PH-HI] File -> Check for newer versions of this program... [/PH-HI] on the menu, or press [PH-HI] Ctrl+U [/PH-HI] on your keyboard.    This feature requires a working connection to the internet.
[PHSECTION] Launching Random Programs
You may launch a random program on your system by clicking [PH-HI] File -> Run... [/PH-HI] on the menu, or press [PH-HI] Ctrl+R [/PH-HI] on your keyboard. 
[PHSECTION] Reporting Problems
You may report problems using this software by clicking [PH-HI] Help -> Send A Bug Report... [/PH-HI] on the menu, or press [PH-HI] F5 [/PH-HI] on your keyboard.  You will be shown detailed instructions to guide you through the short process.
